# Internationalization

Vue for Cesium components are using Chinese **by default**, if you wish you use other
languages, you can get you answer by keep reading.

## Global configuration

Vue for Cesium provides global configurations.

```typescript
import VueCesium from 'vue-cesium'
import enUS from 'vue-cesium/es/locale/lang/en-us'

app.use(VueCesium, {
  locale: enUS
})
```

## ConfigProvider

Vue for Cesium also provides a Vue component [VcConfigProvider](/en-US/component/vc-config-provider)
for globally configuring locale and other settings.

```Vue
<template>
  <vc-config-provider :locale="locale">
    <vc-viewer>
      <vc-navigation></vc-navigation>
    </vc-viewer>
  </vc-config-provider>
</template>
<script setup>
  import { VcConfigProvider, VcViewer, VcNavigation } from 'vue-cesium'
  import enUS from 'vue-cesium/es/locale/lang/en-us'

  const locale = enUS
</script>
```

## CDN Usage

If you are using Vue for Cesium via CDN, then you need to do this, let's again take
unpkg as an example

```html
<script src="//unpkg.com/vue-cesium/dist/locale/zh-cn"></script>
<script>
  app.use(VueMaplibre, {
    locale: VueMaplibreLocaleZhcn
  })
</script>
```

Full documentation refer to: [ConfigProvider](/en-US/component/config-provider)
