# 国际化

Vue for Cesium 组件内部**默认**使用中文，若希望使用其他语言，可以参考下面的方案。

## 全局配置

Vue for Cesium 提供了全局配置国际化的配置。

```typescript
import VueCesium from 'vue-cesium'
import zhCn from 'vue-cesium/es/locale/lang/zh-cn'

app.use(VueCesium, {
  locale: zhCn
})
```

## ConfigProvider

Vue for Cesium 还提供了一个 Vue 组件 [VcConfigProvider](/zh-CN/component/vc-config-provider) 用于全局配置国际化的设置。
用于全局配置国际化的设置。

```Vue
<template>
  <vc-config-provider :locale="locale">
    <vc-viewer>
      <vc-navigation></vc-navigation>
    </vc-viewer>
  </vc-config-provider>
</template>

<script setup>
  import { VcConfigProvider, VcViewer, VcNavigation } from 'vue-cesium'
  import zhCn from 'vue-cesium/es/locale/lang/zh-cn'

  const locale = zhCn
</script>
```

## CDN 用法

如果通过 CDN 使用 Vue for Cesium，那么你需要添加以下代码，以下是使用 unpkg 的示例。

```html
<script src="//unpkg.com/vue-cesium/dist/locale/zh-cn"></script>
<script>
  app.use(VueCesium, {
    locale: VueMaplibreLocaleZhcn
  })
</script>
```

完整文档详见: [ConfigProvider](/zh-CN/component/vc-config-provider)
